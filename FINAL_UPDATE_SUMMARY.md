## 📝 Catatan Penting

### Kenapa Harus Hard Refresh?
Browser menyimpan cache (file lama) untuk loading lebih cepat. Setelah update konten, browser masih menampilkan versi lama dari cache. Hard refresh memaksa browser download file terbaru.

### Jika Masih Muncul Nama Lama
1. Hard refresh: `Ctrl + Shift + R`
2. Clear cache completely: `Ctrl + Shift + Delete`
3. Coba browser lain (Chrome, Firefox, Edge)
4. Coba Incognito/Private mode
5. Restart browser

### File Yang Tidak Perlu Diubah
- CSS files (styling tetap sama)
- JavaScript files (functionality tetap sama)
- Image files (gambar tetap sama)
- Framer runtime files (animation tetap sama)
