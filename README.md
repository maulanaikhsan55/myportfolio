# Maulana Muhammad Ikhsan - Portfolio Website

Personal portfolio website showcasing finance automation systems, ERP integrations, and full-stack development projects.

## 🚀 Tech Stack

- **Framework**: Framer (Exported Static HTML)
- **Deployment**: Vercel
- **Styling**: Embedded CSS with Framer Motion animations

## 📂 Project Structure

```
├── index.html              # Homepage
├── about                   # About page
├── all-projects            # Projects listing
├── contact                 # Contact page
├── project-details/        # Individual project pages
│   ├── tomoro-bridging
│   ├── zeirec
│   ├── api-bank-reconcile
│   ├── duckcost
│   ├── agriduck
│   └── cellupay
├── legal/                  # Legal pages
└── robots.txt              # SEO configuration
```
